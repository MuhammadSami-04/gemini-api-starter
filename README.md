# 🤖 Gemini Prompt Studio

A simple AI chatbot built using Google Gemini API.  
This project allows users to interact with Google's Gemini model using Python in a command-line interface.

---

## 🚀 Features

- Ask questions to Google Gemini AI
- Real-time AI responses
- Secure API key handling using `.env`
- Lightweight and beginner-friendly CLI chatbot

---

## 🧠 Tech Stack

- Python
- Google Generative AI (Gemini API)
- python-dotenv
